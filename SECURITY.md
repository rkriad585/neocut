# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅                 |
| < 1.0   | ❌                 |

## Reporting a Vulnerability

If you discover a security vulnerability in neocut, please report it by
emailing rkriad585@gmail.com.

Do not open a public issue for security vulnerabilities.

You should receive a response within 48 hours. If not, please follow up.

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (if known)

## What to Expect

- Confirmation of receipt within 48 hours
- Assessment and fix timeline within 1 week
- Credit for the discovery (if desired) when the fix is released
